# EverbridgeWCF
Interview Test

1. Build the project  
2. Run EverbridgeWCF  
3. Run EverbridgeClientSide Console App  
4. Info will be printed to the Console  

Press Number to choose the action:
- 1 - Get all
- 2 - Open
- 3 - Close
- 4 - Lock
- 5 - Unlock
- 6 - Get status
- 7 - Add new
- 8 - Change label
- 9 - Remove
- 0 - Print Instruction
